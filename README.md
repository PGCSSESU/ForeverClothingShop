## ü™ú Application Flow (Step-by-Step)

1. User lands on the home page
2. Latest product collection loads with entrance animation
3. User scrolls through animated product cards
4. Hovering over a product reveals additional details
5. User clicks a product card
6. Product details page opens
7. User adds product to cart
8. Cart count updates in real-time
9. User reviews cart items
